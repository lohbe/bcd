# Part 1 — Identifying a Personal Behaviour & Setting Goals

> Target length: **400–500 words**.

## Target behaviour

Establish a habitual coding practice: a daily, time-boxed coding session built around
worked examples.

## Why this behaviour is meaningful to me / alignment with values & long-term goals

<!-- ~80-100 words. -->
- Develop computational, graph, and functional thinking.
- Build programming skills for personal interest and future career growth.
- Replace "chaotic debugging / procrastination / imperative thinking" with deliberate,
  systematic practice.

## Positive impact on my life

<!-- ~60-80 words. -->
- When I want to study, practice coding, or think systematically, I often fall back on
  old habits (chaotic debugging, procrastination, imperative thinking). A habitual
  practice gives me a reliable, low-friction on-ramp to the mode of working I actually
  value.

## S.M.A.R.T. goal

| Element | Detail |
|---|---|
| **Specific** | Initiate a 25-minute coding sprint or a worked example. |
| **Measurable** | Timestamp of the first git commit of the day. |
| **Achievable** | 25 minutes is small enough to start daily; this repo auto-records it. |
| **Relevant** | Builds the computational/graph/functional thinking I value. |
| **Time-bound** | 21 days, 2026-07-01 → ~2026-07-21, within an 8–10am window. |

> **Goal statement:** I will initiate a 25-minute coding sprint or worked example between
> 8–10am daily for 21 days, measured by the timestamp of the first commit of the day.

## COM-B analysis of early challenges / supports

<!-- ~120-150 words across the three components. -->

**Capability**
- *Facilitator*: existing coding skills reduce the activation cost.
- *Challenge*: worked examples require choosing a concept (cognitive load); mitigate by
  preparing the next day's topic the evening before.

**Opportunity**
- *Facilitator*: fixed 8–10am window + pre-set-up environment (the repo, editor open).
- *Challenge*: morning interruptions; mitigate by committing before checking messages.

**Motivation**
- *Facilitator*: career-growth and intrinsic interest (reflective); habit stacking onto an
  existing morning cue (automatic).
- *Challenge*: low morning motivation; mitigate via implementation intentions
  ("After I pour coffee at 08:00, I open the editor and start the sprint").
